# PokeMuro

Aplicacion React creada con Vite para practicar Hooks, Axios, servicios,
componentes y formularios controlados.

La app consulta publicaciones con Axios desde JSONPlaceholder y muestra una
lista adaptada como publicaciones de una comunidad de entrenadores Pokemon.
Tambien permite crear publicaciones, eliminarlas, filtrarlas y validar el
formulario.

## Tecnologias

- React
- Vite
- JavaScript
- Axios

## Instalacion

```bash
npm install
```

## Ejecutar el proyecto

```bash
npm run dev
```

Luego abra la URL indicada por Vite, normalmente:

```bash
http://127.0.0.1:5173/
```

En PowerShell de Windows puede usar:

```bash
npm.cmd run dev
```
